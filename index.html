Patient Finder / API

This application is a simple interface to access a separate near-RESTful API. Users are forced to sign in to a Google Account, and are then able to Add or Search for patients in the database. Only administrators to the site are able to Delete from the database.

The application was built on the Google App Engine, using Python, Javascript, HTML, CSS and Bootstrap.

How-To Election Map

I wrote a how-to guide to using the Leaflet.js javascript library, as well as GeoJSON data, to create an election map. The guide discusses how to use geolocation in the browser to set a marker on the user's current location. In addition, the map allows the user to tap and hold on a state to reveal the results from the 2012 presidential election.
