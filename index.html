I recently graduated with a bachelor's degree in computer science from Oregon State University. For the last couple years, I worked as a journalist at the Lincoln Journal Star, the Omaha World-Herald and The Oregonian. Now, I'm interested in technology and software development.

Class and personal projects

Graphical Web Crawler

I designed and built a graphical web crawler application with two classmates for a Software Projects course. The user enters a starting URL, as well as the number of links they wish to crawl, and hits submit. The application then returns a tree-like visualization of the crawl path.

The crawler was built using Python, and the front-end visualization was built using D3.js. The application was built on top of the Google App Engine.

Business Jargon Website

This site was an experiment in working with Parallax.js and ScrollReveal.js. The site, built on the Google App Engine, also makes heavy use of Bootstrap. The project was adapted from the Pooper App, a fake app that has a really sleek web design. I wanted to see how it worked, so I created this site for a fake business, BBB LLC, and filled the sections with business jargon.

Workout Tracker

For a mobile and cloud computing class, I created this simple workout tracker application to test out the Google Datastore NoSQL database. The user is able to add a workout routine, view their routines and edit individual routines easily.

The application was built with Bootstrap, HTML, CSS and Python, on top of the Google App Engine.

Twitter Bot

This twitter bot represents the bus stop near my apartment in Portland, and tweets when the 15 and 20 bus lines are arriving. The bot was built with Python and the Tweepy library, using an API ran by my transit agency, TriMet. The bot runs on Heroku.

Election Map

I created a state-level election map, as well as a how to guide, using Leaflet.js. I love data and maps, so Leaflet.js was a very interesting library to learn about. The assignment called for the library to be intended for mobile use, so I tried hard to manipulate the Leaflet.js library into some mobile friendly. As a result, the user must tap and hold onto a state in order to see the election data.

Usability: Point of Sale App

Using prototyping tool Balsamiq, a group of classmates and I created a wireframe for a check-splitting app to be used by both diners and restaurants. The version of the app used by diners would allow them to split their bill with other diners as they wish, as well as add a tip, with a few taps. We focused on making the application quick, easy to use and usable.

For a databases course, I designed a SQL relational database for biographical presidential data. For the project, I created an ER diagram, a schema, a list of table creation queries and a simple application built with AJAX, node.js and HTML to allow a user to add data to the database.

Angular.js Inbox

After reading up on Angular.js, I decided to create a tutorial app, authored by Thinkful.com, to get the hang of the syntax. This inbox application draws email messages from a JSON file, and allows the user to read and delete the messages.
